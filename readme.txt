Displays a random cat fact!

== Description ==

This plugin generates a random cat fact and displays it using a shortcode or php function that can be inserted in your WordPress theme.

== Installation ==

1. Upload `cat_facts.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How do I use this plugin? =

to display a random cat fact inside a post or page, use the shortcode [catfacts]. To integrate a cat fact in your theme files, you can use the function cat_facts();

= Can I use this on a commercial site? =

This plugin was intended for non-commercial sites only.
